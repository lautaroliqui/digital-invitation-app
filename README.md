# Invitación Interactiva - Evento Privado

Una Single Page Application (SPA) estática, liviana y responsiva desarrollada para funcionar como una invitación digital interactiva. Optimizada para dispositivos móviles con un enfoque estricto en el rendimiento y la experiencia de usuario (UX).

## 🚀 Tecnologías Utilizadas

*   **HTML5 Semántico:** Estructuración accesible de la información.
*   **CSS3:** Sistema de grillas (CSS Grid) y Flexbox para diseño responsivo. Variables globales para control de temas y animaciones CSS puras para efectos visuales (hojas cayendo) sin impacto en el rendimiento.
*   **Vanilla JavaScript:** Lógica de cuenta regresiva asíncrona y manipulación de la API de HTMLAudioElement para la creación de un reproductor de música customizado.
*   **SVG Line Art:** Iconografía generada y manipulada vectorialmente para asegurar nitidez absoluta en pantallas retina sin consumo excesivo de ancho de banda.

## ⚙️ Características Técnicas

*   **Zero Dependencies:** El proyecto no utiliza librerías externas (ni React, ni Tailwind, ni jQuery), garantizando un tiempo de carga casi instantáneo (Time To Interactive mínimo).
*   **Gestión de Estado de Audio:** Controles interactivos que sincronizan la reproducción multimedia con las animaciones del DOM.
*   **Integración de API de Terceros:** Redireccionamiento dinámico con parámetros URL codificados para la API de WhatsApp (`wa.me`).

## 🛠️ Instalación y Uso Local

Al ser un sitio completamente estático, no requiere un proceso de build.

1. Clonar el repositorio:
   \`git clone https://github.com/TU_USUARIO/invitacion-santy-18.git\`
2. Abrir \`index.html\` en cualquier navegador web moderno o servirlo mediante Live Server.
